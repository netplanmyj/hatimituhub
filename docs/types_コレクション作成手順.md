# Firestore typesコレクション作成手順

## 目的
区分コードと区分名の対応表を管理するため、`types` コレクションを作成します。

## 作成手順
1. Firebaseコンソールで「Firestore Database」を開く
2. 「コレクションを開始」ボタンをクリック
3. コレクションIDに `types` を入力
4. ドキュメントIDに区分コード（例: 1, 2, 3, ...）を入力
5. フィールドを追加
   - `typeLabel`（文字列）: 区分名（例: "卸"）
6. 「保存」

## 登録例
| ドキュメントID | typeLabel   |
|----------------|-------------|
| 1              | 卸          |
| 2              | イベント    |
| 3              | 電話        |
| 4              | リーフレット |
| 5              | 47クラブ    |
| 6              | 送料        |

---

この手順で区分情報を `types` コレクションに登録できます。アプリ側で `products.type` を使って区分名を参照できます。
