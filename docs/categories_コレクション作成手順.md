# Firestore categoriesコレクション作成手順

## 目的
商品種別（カテゴリ）のコードと名称の対応表を管理するため、`categories` コレクションを作成します。

## 作成手順
1. Firebaseコンソールで「Firestore Database」を開く
2. 「コレクションを開始」ボタンをクリック
3. コレクションIDに `categories` を入力
4. ドキュメントIDに種別コード（例: 1, 2, 3, ...）を入力
5. フィールドを追加
   - `categoryLabel`（文字列）: 種別名（例: "春の百花蜜"）
6. 「保存」

## 登録例
| ドキュメントID | categoryLabel         |
|----------------|----------------------|
| 1              | 春の百花蜜           |
| 2              | みかん蜜             |
| 3              | ハゼノ木の蜜         |
| 4              | 中島四季の蜜〜初夏〜 |
| 5              | アカメガシワの蜜     |
| 6              | 中島四季の蜜〜夏季〜 |
| 7              | マサキの蜜           |
| 8              | 久万高原町 山の蜜    |
| 9              | 組合せ品             |

---

この手順で商品種別情報を `categories` コレクションに登録できます。アプリ側で `products.category` を使って種別名を参照できます。
